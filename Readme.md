# Программа для подсчета единичных бит
---
Консольная программа для подсчета единичных бит в файле.
Для запуска программы стоит написать 
```make```
в папке src проекта.

# Тесты
---
Для тестирования на проверку значений используется файл test.sh. Для тестирования значений используйте команду
```make test```
Для тестирования на время используется test_time.sh. При проверки на время программа берет файлы из папки files. Для тестирование на время выполнения используйте команду
```make time```
Тестирование программы проводилось на компиляторе gcc 15.2.1, ОС: Arch Linux

Для очистки проекта используйте команду
```make clean```

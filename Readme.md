# Программа для подсчета единичных бит
---
Консольная программа для подсчета единичных бит в файле.

# Тесты
---
Для тестирования на проверку значений используется файл test.sh.
Для тестирования на время используется test_time.sh. При проверки на время программа берет файлы из папки files.

Тестирование программы проводилось на компиляторе gcc 15.2.1, ОС: Arch Linux
